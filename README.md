# 🌿 Mon Jardin

Application de suivi de plantes avec IA (Claude).

## Installation

```bash
npm install
npm run dev
```

## Déploiement sur Vercel

1. Crée un compte sur [vercel.com](https://vercel.com)
2. Installe Vercel CLI : `npm i -g vercel`
3. Lance : `vercel deploy`

Ou directement depuis GitHub :
1. Pousse le code sur GitHub
2. Importe le repo sur vercel.com
3. C'est tout !

## Clé API

L'app te demandera ta clé API Anthropic au premier lancement.
Obtiens-en une sur : https://console.anthropic.com

La clé est stockée uniquement sur ton appareil (localStorage).
